Multiple Discounts in Purchase.


In many cases we need to apply Discount on discount but normal odoo does not support that so to have such functionality we have added this module. Using this module you can add discount on discount in Sale and Purchase lines. In Sale and Purchase Lines you can add multiple discount amounts separated by '+'. If you apply '10+5+2' it will apply first 10% discount and then it will apply 5% on new amount and then it will apply 2%.

Option to add multiple discounts in Purchase and Print amounts n report.
